What are schemes?

  • A cipher scheme is the method used for encrypting a text/string by using a set of key-values set for that scheme. Cipher-Bay API uses schemes to encode and decode text. Different shemes have different keys and values set to encode and decode your string/text accurately.

  • The currently available schemes on Cipher-Bay API can be viewed . You can choose any of these schemes to encode your text.

  • The text/string is decoded with the same scheme which was used to encode them to get the desirable output. However, you don't need to remember the scheme you used to encode a string, the encoded cipher has a scheme flag associated with it for that recognition.

  • The schemes come with pre set alias to make them easier to use. Some examples of aliases are :
    scheme_1 (scheme) alpha (alias)
    scheme_2 (scheme) beta (alias)
    scheme_3 (scheme) gamma (alias)
